EGR304 Software Install (Fall 2023)

By ESD Faculty

Introduction

You will need to install all of the software listed below on any personal computers you plan to use this semester. Most of the software below is for Windows. If you have a Mac, you can use Parallels to run Windows apps.

1. Discrete Installers

PSoC® Creator™ (Windows only)

  1. Create an account with Infineon
  2. Download and install PSoC Creator
    1. Choose Typical Installation
  3. Once the install completes, select run updater (help–>update manager) from PSoC Creator and update all packages

CySmart Bluetooth Tool (Windows only)

  • AIROC CySmart 1.3

LightBlue® Phone App

  • Apple App Store version

  • Google Play Store version

2a. Additional Installers via Chocolatey (optional, but handy)

Chocolatey is a Windows-based package manager that automates downloading and installing software for Windows.

  1. Open up PowerShell (Start –> “Windows Powershell”) in administrator mode (click on the arrow to the right of “Windows Powershell” text in the start menu and “run as administrator”)

  2. Paste in the following line to install chocolatey

    Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))

  3. Close the powershell window and open a new powershell window (also in administrator mode). Paste in the following line:

    choco install -y putty.install kicad vcredist2013

  4. Paste in the following line for some optional but very useful tools:

    choco install -y adobereader notepadplusplus 7zip grepwin kdiff3

2b. Additional Installers - Manual Installation

If you did not follow the chocolatey install instructions above, then manually install the following apps:

PuTTY Terminal Programs (Windows only)

https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html

KiCad (Windows / Mac / Linux)

https://www.kicad.org/download/

Visual C++ Redistributable (Windows only)

http://updates.cypress.com/updates/prerequisite/vcredist_x86_2013.exe

3. Configure VB 3.5

  1. Go to Start –> Control Panel –> Programs
  2. On the left side bar, select “Turn windows features on or off”
  3. Select “.NET framework 3.5”
